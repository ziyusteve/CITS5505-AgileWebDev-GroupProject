Supported formats: CSV, TXT, XLSX, JSON
Upload Instructions

CSV (Comma-Separated Values) files should:

  • Have a header row with column names
  • Use comma (,) as the separator
  • Use consistent data types within each column
date,temperature,humidity
2025-01-01,22.5,65
2025-01-02,23.1,67
2025-01-03,21.8,70

Excel (.xlsx) files should:

  • Have a header row with column names
  • Data should start from the first sheet
  • Avoid merged cells for best results

JSON files should:

  • Be properly formatted valid JSON
  • Preferably contain an array of objects for tabular analysis
[
  {"date": "2025-01-01", "temperature": 22.5, "humidity": 65},
  {"date": "2025-01-02", "temperature": 23.1, "humidity": 67},
  {"date": "2025-01-03", "temperature": 21.8, "humidity": 70}
]
